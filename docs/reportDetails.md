# 实验评分标准与提交要求

## 实验评分标准

项目难度分 = Min(110, 项目基础分+项目扩展分)

项目完成分 = 需求分析说明书（10分）+
系统设计文档（20分）+
软件测试报告（20分）+
AI辅助编程学习证书（10分）+
源码质量（10分）+
项目答辩（10分）+
项目演示（20分）

小组得分 = 项目完成分*项目难度分/100

组长得分 = Min(100, 小组得分)
组员得分 = Min(100, 小组得分-5)


## 提交资料

以小组为单位：
- 项目报告文档
- 项目说明PPT
- AI辅助编程学习证书
- 源代码+可执行程序（搭配shell、makefile、Cmakelist、dockerfile等）
- 项目演示视频（2分钟以内）

## 提交方式

1-3：提交到实验平台（单独的pdf文档，打包提交）；
4-5：打包提交到助教邮箱。

## 截止日期
最后一次实验课（第十次课）。
