# 子系统七：文件系统抽象用例描述

参与者：BackupEngine、RestoreEngine、Packer

---

## UC-FS-01：递归遍历目录树

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-FS-01 |
| **用例名称** | 递归遍历目录树 |
| **参与者** | BackupEngine/RestoreEngine |
| **前置条件** | 源路径存在且为目录 |
| **后置条件** | 返回包含所有文件和子目录的 vector<FileEntry>，每项含相对路径、类型、元数据 |
| **基本事件流** | ① `LocalFsAbstraction::walk(root)` → ② `std::filesystem::recursive_directory_iterator` → ③ 对每个 entry 调用 `detectFileType()` → ④ 调用 `readMetadata()` → ⑤ 对符号链接读 target → ⑥ 对设备文件读 major/minor → ⑦ 收集到 vector |
| **备选事件流** | 3a. 遇权限不足的子目录 → 跳过并返回 kPermissionDenied；3b. 遇 symlink loop → 返回 kSymlinkLoop |
| **包含/扩展** | 包含 UC-FS-08（检测类型）、UC-FS-09（读元数据） |
| **优先级** | 高 |

## UC-FS-02：读取文件内容

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-FS-02 |
| **用例名称** | 读取文件内容 |
| **参与者** | BackupEngine/Packer |
| **前置条件** | 文件存在、可读 |
| **后置条件** | 返回完整的文件内容 vector<char> |
| **基本事件流** | ① `LocalFsAbstraction::read(path)` → ② ifstream binary → ③ file_size 开缓冲区 → ④ read 全部内容 |
| **备选事件流** | 2a. 文件不存在 → 返回 kPathNotFound |
| **包含/扩展** | 无 |
| **优先级** | 高 |

## UC-FS-07：创建设备节点 (block/char)

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-FS-07 |
| **用例名称** | 创建设备节点 (block/char) |
| **参与者** | BackupEngine/RestoreEngine |

## UC-FS-08：检测文件类型

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-FS-08 |
| **用例名称** | 检测文件类型 |
| **参与者** | BackupEngine（walk 中） |
| **前置条件** | 路径有效 |

| **用例编号** | UC-FS-09 |
| **用例名称** | 读取文件元数据 |
| **参与者** | BackupEngine（walk 中） |

## UC-FS-10：恢复文件元数据

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-FS-10 |
| **用例名称** | 恢复文件元数据 |
| **参与者** | BackupEngine/RestoreEngine |
| **前置条件** | 文件/目录已在目标路径创建 |
| **后置条件** | 属主/权限/时间戳已恢复 |
| **基本事件流** | ① `restoreMetadata(path, meta, restoreOwnership)` → ② `lchown(path, uid, gid)` 恢复属主 → ③ `fchmodat(AT_FDCWD, path, mode, 0)` 恢复权限（Linux 对 symlink 跳过） → ④ `utimensat(AT_FDCWD, path, times, AT_SYMLINK_NOFOLLOW)` 恢复时间戳 |
| **备选事件流** | 2a. restoreOwnership=false 或非 root → 跳过 lchown |
| **包含/扩展** | 包含 UC-FS-11（检测是否可恢复属主） |
| **优先级** | 高 |
