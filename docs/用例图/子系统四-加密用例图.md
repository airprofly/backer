# 子系统四：加密用例描述

参与者：handleBackup/handleRestore（CLI 命令处理器）、用户（密码输入者）

---

## UC-EN-01：选择加密算法

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-EN-01 |
| **用例名称** | 选择加密算法 |
| **参与者** | handleBackup/handleRestore |
| **前置条件** | 用户已通过 --encrypt/--decrypt 参数指定算法 |
| **后置条件** | `buildEncryptor(name)` 创建 OpenSslEncryptor 实例 |
| **基本事件流** | ① CLI 解析 --encrypt aes256|sm4 → ② validateAlgo 校验 → ③ `buildEncryptor` → ④ OpenSslEncryptor 构造时选择 EVP_CIPHER |
| **备选事件流** | 1a. 算法名无效 → CLI 报错 |
| **包含/扩展** | 被 UC-EN-05/06/07/08 包含 |
| **优先级** | 高 |

## UC-EN-03：交互式输入密码

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-EN-03 |
| **用例名称** | 交互式输入密码 |
| **参与者** | 用户 |
| **前置条件** | --password 参数未提供但 --encrypt/--decrypt 已指定 |
| **后置条件** | 密码从终端不回显地读入 |

| **包含/扩展** | 扩展 UC-EN-05（加密时）或 UC-EN-07（解密时） |
| **优先级** | 中 |

## UC-EN-05：AES-256-GCM 加密

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-EN-05 |
| **用例名称** | AES-256-GCM 加密 |
| **参与者** | handleBackup |
| **前置条件** | 已完成打包+压缩（可选），当前有归档文件待加密 |
| **后置条件** | 文件后缀追加 `.enc`；文件内容包含 EncryptedHeader + 密文 + GCM 认证标签 |
| **基本事件流** | ① `encryptFile(inFile, outFile, enc, password)` → ② 读入完整文件 → ③ `OpenSslEncryptor::encrypt()` → ④ 随机 salt/IV 生成 → ⑤ PBKDF2-SHA256 派生 256-bit 密钥 → ⑥ EVP_EncryptInit_ex with AES-256-GCM → ⑦ EVP_EncryptUpdate/EncryptFinal → ⑧ 前置写入 EncryptedHeader → ⑨ 写入密文 → ⑩ 写入 GCM 认证标签(16B) → ⑪ `OPENSSL_cleanse` 清理密钥 |
| **备选事件流** | 3a. OpenSSL EVP 初始化失败 → 返回 kEncryptionFailed |
| **包含/扩展** | 包含 UC-EN-01/09/10/12/13；扩展 UC-EN-02/03/04/11 |
| **优先级** | 高 |

## UC-EN-07：AES-256-GCM 解密

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-EN-07 |
| **用例名称** | AES-256-GCM 解密 |
| **参与者** | handleRestore |
| **前置条件** | 备份源为 .enc 文件；--decrypt aes256 指定 |
| **后置条件** | 解密后的临时文件用于后续解压/解包；解密失败时无文件输出 |
| **基本事件流** | ① `decryptFile(inFile, outFile, enc, password)` → ② 读入完整 .enc 文件 → ③ 解析前 47 字节 EncryptedHeader（magic+algo+salt+iterations+iv） → ④ PBKDF2 重派生密钥 → ⑤ EVP_DecryptInit → ⑥ 解密 → ⑦ 验证 GCM 认证标签 → ⑧ 写入解密后的临时文件 |
| **备选事件流** | 3a. magic 不是 "BACKERENC\0" → 报错；4a. 密码错误 → GCM 标签校验失败 → 返回 kEncryptionFailed → 不输出文件 |
| **包含/扩展** | 包含 UC-EN-09/11/12 |
| **优先级** | 高 |

## UC-EN-09：密钥派生 (PBKDF2)

| 字段 | 内容 |
|------|------|
| **用例编号** | UC-EN-09 |
| **用例名称** | 密钥派生 (PBKDF2) |
| **参与者** | Encryptor |
| **前置条件** | 已有 password 和 salt |

| **包含/扩展** | 被 UC-EN-05/06/07/08 包含 |
| **优先级** | 高 |
