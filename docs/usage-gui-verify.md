## 1. 环境检查与启动

### 1.1 检查 GUI 运行环境

```bash
# 1. 确认有图形桌面环境
echo $DISPLAY          # 应输出 :0 或 :1（桌面）或为空（无桌面）
which Xvfb             # Xvfb 可用于无桌面环境

# 2. 确认 GUI 已编译
ls -lh build/backer-gui    # 应存在，约 5MB

# 3. 生成测试数据（如还未生成）
bash scripts/setup-testdata.sh
```

> **无桌面环境？** 安装 Xvfb：`sudo apt install xvfb && xvfb-run ./build/backer-gui`

### 1.2 启动 GUI

方式一（推荐）：

```bash
bash scripts/run-gui.sh
```

方式二（直接启动）：

```bash
./build/backer-gui
```

方式三（无桌面）：

```bash
xvfb-run ./build/backer-gui
```

### 1.3 错误排查

| 启动问题 | 原因 | 解决 |
|---------|------|------|
| `cannot connect to X server` | 无桌面环境 | 安装 Xvfb 或用桌面 Linux |
| `libEGL warning` | 缺少 3D 加速 | 忽略，或 `export LIBGL_ALWAYS_SOFTWARE=1` |
| `QFont::setPixelSize: Pixel size <= 0` | 字体配置 | 忽略，或 `sudo apt install fonts-noto-cjk` |
| 程序立即退出无窗口 | Qt 平台问题 | `export QT_DEBUG_PLUGINS=1` 查看详细日志 |

│   功能区域（因标签页而异）                         │

### 2.1 备份标签页布局

| 区域 | 控件 | 说明 |
|------|------|------|
| 源目录 | 文本输入框 + 浏览按钮 | 选择要备份的目录 |

| 筛选 | 复选框 + 编辑按钮 | 打开筛选对话框（6 维度） |
| 底部 | 开始备份 / 取消 / 恢复默认 | 操作按钮 |
