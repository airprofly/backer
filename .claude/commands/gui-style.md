# GUI 设计风格：macOS 简约

## 设计语言

受 macOS 系统原生界面启发，采用简洁、通透、注重内容的设计风格。

### 核心理念

- **通透留白**：大量留白，不用边框分隔，用间距建立层次
- **扁平克制**：扁平 UI，无强烈阴影，无渐变，无冗余装饰
- **内容优先**：UI 退居幕后，突出数据展示和操作
- **一致细腻**：统一的 8px 栅格系统，微交互动效

### 色板

```
背景主色:     #F5F5F7   (macOS 标准浅灰)
面板/卡片色:  #FFFFFF   (纯白)
侧栏/分割线:  #E8E8ED   (浅灰分割)
悬浮高亮:     #E8E8ED

强调色(蓝):   #007AFF   (macOS 标准蓝)
成功色(绿):   #34C759   (macOS 标准绿)
警告色(橙):   #FF9500   (macOS 标准橙)
错误色(红):   #FF3B30   (macOS 标准红)

文字主色:     #1D1D1F   (几乎黑)
文字次要:     #86868B   (浅灰)
文字占位:     #C7C7CC

窗口按钮红:   #FF5F57
窗口按钮绿:   #28C840
窗口按钮橙:   #FEBD2E
```

### 字体

```
西文: -apple-system, BlinkMacSystemFont, "Segoe UI", "Helvetica Neue"
中文: "PingFang SC", "Noto Sans CJK SC", "Microsoft YaHei"
等宽: "SF Mono", "JetBrains Mono", "Cascadia Code"
```

### 圆角与间距

```
小圆角: 6px   (按钮、输入框)
中圆角: 8px   (卡片、面板)
大圆角: 12px  (弹窗、对话框)

间距基准: 8px 栅格系统 (8/16/24/32/40)
```

### 按钮规则

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| **强调按钮** | 蓝色填充(#007AFF) + 白色文字 + 圆角 6px | 主要操作：开始备份、保存 |
| **默认按钮** | 白色背景 + 浅灰边框(#D2D2D7) + 圆角 6px | 次要操作：取消、浏览 |
| **文字按钮** | 无背景 + 蓝色文字 | 轻量操作：编辑筛选、+/- |
| **危险按钮** | 红色文字 | 删除操作 |

### 输入框

```
背景: #FFFFFF
边框: #D2D2D7 (1px solid)
聚焦: #007AFF 边框
圆角: 6px
内边距: 8px 12px
```

### 标签页 (Tab)

```
macOS 风格分段控件 (Segmented Control)：
- 背景: #E8E8ED
- 选中: #FFFFFF + 微阴影
- 圆角: 8px
- 间距: 4px 内边距
```

### 进度条

```
纤细型，高度 4px
轨道: #E8E8ED
填充: #007AFF
圆角: 2px
```

### 表格/列表

```
无边框，交替行背景
表头: #F5F5F7 + 次要文字
行高: 32px
选中行: #E8E8ED 背景
```
